Описание FileManager

При запуске приложение задает себе размер и цвет.
Открывается окно с папками и файламы в текущей папке.
Вызывается окно информации (InformationWindow).
У пользователя запрашивается команда и сравнивается со списком команд
Если команда не найдена, вызывается метод Recognitio для распознования более сложных команд
Если команда не распознана выводится окно с сообщением.
Команды предусмотренные для пользователя:
*названия файлов и папок в командах записываются между /file/
next  следующая страница
back  предыдущая страница
cd //  перемиститься в папку
cd ..  выйти из папки
copy /копируемый файл/ /копия/  копирование файла
delete //  удаление файла
info //  информация о файле
help  окно со списком команд
Ошибки приложения записываются в текстовый файл
Движение по истории команд по умолчанию есть в консоли
